# Sales & Inventory Management System (LIVE)

Full-stack enterprise app for retailers.

## Tech Stack
- **Backend**: .NET 8 Web API + EF Core + SQL Server
- **Frontend**: Angular 18 + Material UI + RxJS
- **Auth**: JWT + Role-based
- **Deploy**: Render.com (Backend) | Vercel (Frontend)

## Features
- Real-time stock update
- GST/Non-GST billing
- Low stock alerts
- Admin + User dashboard
- PDF Reports
